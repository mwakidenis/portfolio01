## üöÄ Getting Started

Follow these steps to run the project locally:

### Step 1: Clone the repository
```bash
git clone https://github.com/madebydenis/portfolio-repo.git
```

### Step 2: Navigate to the project folder
```bash
cd portfolio-repo
```

### Step 3: Open in browser
- **Option 1**: Double-click `index.html` in your file explorer
- **Option 2**: Use a local server for better experience:
  ```bash
  # Using Python
  python -m http.server 8000

  # Using Node.js (if available)
  npx serve .
  ```

### Step 4: View the portfolio
The site will be available at `http://localhost:8000` (if using server) or directly in your browser.

## üé® Customization Guide

### Personal Information
Update the hero section with your details:
```html
<p>Name: <span class="text-cyan-500 font-bold">Your Name</span></p>
<p>Education: <span class="text-cyan-500 font-bold">Your Degree</span></p>
<p>Profession: <span class="text-cyan-500 font-bold">Your Title</span></p>
```

### Color Scheme
Modify the gradient in CSS:
```css
body {
  background: linear-gradient(135deg, #your-color1 0%, #your-color2 100%);
}
```

### Skills Section
Add or modify skill cards:
```html
<div class="skill-card rounded-xl p-8 shadow-xl text-center">
  <i class="fas fa-your-icon colored-icon text-your-color text-7xl mb-6 bounce-slow"></i>
  <h3 class="text-2xl font-semibold text-your-color mb-4">Your Skill</h3>
  <p class="text-gray-300">Description of your expertise</p>
</div>
```

### Social Links
Update social media URLs:
```html
<a href="https://your-profile-link" target="_blank" class="text-4xl hover:text-cyan-400 transition-all duration-300 colored-icon hover:scale-125">
  <i class="fab fa-your-platform"></i>
</a>
```
